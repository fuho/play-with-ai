Data Management

Instructions

  • Use +/- buttons or Mouse Wheel on canvas to Zoom. Middle Mouse/Alt+Drag to Pan.
  • Set Grid Scale.
  • Use Tools:
    • Select/Move (Default): Click & drag any point (origin, outline, measurement).
    • Set Origin: Click to place origin.
    • Draw Outline: Click to add vertices. Right-click vertex to delete. Click start vertex to close.
    • Add/Edit Measurements: Click empty spot to add (prefills last height). Click existing point to edit height. Right-click point to delete. Use Mouse Wheel over point to adjust height.
  • Adjust Interpolation method & Contour Interval.
  • Check desired Visualizations (can be layered).
  • Right-click any point (origin, outline, measurement) to delete it, regardless of selected tool.
  • Copy State / Paste & Import State / Reset to Default.
